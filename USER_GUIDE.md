## üìñ Using Features

### FMA Municipalities

**Purpose**: View comprehensive list of municipalities covered by FMA 6 and FMA 9.

**How to Use**:

1. **Navigate to FMA Municipalities**:
   - Click "FMA Municipalities" in the navigation bar
   - Or use the quick link on the home page

2. **Filter Data**:
   - **Filter by FMA**: Select a specific FMA (6 or 9) or view all
   - **Filter by Region**: Select a region to narrow results
   - **Filter by Province**: After selecting a region, choose a province
     - Note: Province dropdown is enabled only after selecting a region

3. **Navigate Large Datasets**:
   - Use the "Items per page" dropdown (10, 20, 50, 100)
   - Use pagination controls at the bottom to move between pages
   - View pagination info showing current range (e.g., "Showing 1-20 of 150")

4. **View Summary Statistics**:
   - Check the sidebar for:
     - Total number of municipalities
     - Number of NSAP monitored municipalities
     - Municipalities by Region (alphabetically sorted, BARMM at bottom)
     - Municipalities by FMA (numerically sorted)

5. **Understanding the Table**:
   - **FMA Column**: Shows the FMA identifier (FMA 6 or FMA 9)
   - **Region Column**: Geographic region
   - **Province Column**: Province name
   - **City/Municipality Column**: City or municipality name

### Landing Centers Map

**Purpose**: Visualize landing centers on an interactive map with filtering capabilities.

**How to Use**:

1. **Navigate to Landing Centers**:
   - Click "Landing Centers" in the navigation bar
   - Or use the quick link on the home page

2. **Access Filters**:
   - The filter sidebar is visible by default on desktop
   - On mobile, click the filter toggle button (funnel icon) to show/hide filters
   - Click the "X" button to close the filter sidebar

3. **Apply Filters**:
   - **Filter by FMA**: Select FMA 6 or FMA 9
   - **Filter by Region**: Select a specific region
   - **Filter by Province**: Select a province (enabled after selecting region)
   - **Filter by Fishing Ground**: Select a fishing ground
   - **Search City/Municipality**: Type to search by city or municipality name
   - **Search Landing Center**: Type to search by landing center name
   - Filters update the map markers in real-time

4. **Use Map Controls**:
   - **Zoom**: Use mouse wheel, +/- buttons, or pinch gesture on mobile
   - **Pan**: Click and drag to move around the map
   - **Base Map Control**: Click the layer icon (top-left) to switch between:
     - OpenStreetMap (default)
     - Satellite view
     - Other available map layers

5. **View Marker Information**:
   - Click on any marker to see details about the landing center
   - The info panel shows the count of visible markers
   - Only locations with valid coordinates are displayed

6. **Reset Filters**:
   - Click "Reset All Filters" button to clear all filters
   - Map will show all landing centers again

7. **Mobile Tips**:
   - Use the filter toggle button to maximize map viewing area
   - Pinch to zoom on the map
   - Tap markers to view information

### Activities

**Purpose**: Browse timeline of activities, trainings, and coordination meetings.

**How to Use**:

1. **Navigate to Activities**:
   - Click "Activities" in the navigation bar

2. **Filter Activities**:
   - **By Year**: Select a specific year from the dropdown
   - **By Search**: Type keywords to search by:
     - Activity title
     - Location
     - Resource person name

3. **View Activity Details**:
   - Each activity card shows:
     - Date conducted
     - Activity title
     - Location
     - Resource person
     - Reference documents (if available)

4. **Timeline View**:
   - Activities are displayed in chronological order
   - Most recent activities appear first
   - Cards are visually connected in a timeline layout

### Directory

**Purpose**: Access contact information for project personnel.

**How to Use**:

1. **Navigate to Directory**:
   - Click "Directory" in the navigation bar

2. **Switch Between Directory Types**:
   - Use tabs to switch between:
     - **Internal**: NFRDI FishCore Personnel
     - **External**: External partners and stakeholders
     - **NPMO**: National Project Management Office personnel

3. **Filter Directory Entries**:
   - **Internal Directory**:
     - Filter by Component
     - Filter by Employment Type
     - Search by name, position, or email
   - **External Directory**:
     - Filter by Office
     - Filter by FMA Lead
     - Search by name, position, or email
   - **NPMO Directory**:
     - Filter by Component
     - Filter by Office
     - Search by name, position, or email

4. **Export Data**:
   - Click "Download as CSV" button
   - The exported file includes all currently filtered results
   - Open the CSV file in Excel, Google Sheets, or any spreadsheet application

5. **View Contact Details**:
   - Each entry shows:
     - Full name
     - Position/Designation
     - Component/Office
     - Email address
     - Additional information based on directory type

### References

**How to Use**:

## üîç Search and Filtering

### Global Search

**How to Use**:

1. **Access Search**:
   - Click the search bar in the navigation (top right)
   - Or navigate to the Search page

2. **Enter Search Query**:
   - Type your search term
   - Press Enter or click the search icon

3. **View Results**:
   - Results are grouped by page/section
   - Each result shows:
     - Page title
     - Matching items
     - Relevance score
   - Click "View Page" to navigate to the relevant section

4. **Recent Searches**:
   - Previously searched terms are saved
   - Click on a recent search to repeat it

### Filtering Best Practices

1. **Start Broad, Then Narrow**:
   - Begin with general filters (e.g., FMA)
   - Then apply specific filters (e.g., Region, Province)

2. **Combine Filters**:
   - Multiple filters work together
   - Use "Reset" buttons to clear all filters at once

3. **Use Search Within Filters**:
   - Text search works alongside dropdown filters
   - Search is case-insensitive

4. **Check Summary Statistics**:
   - In FMA Municipalities, use the summary sidebar to understand data distribution
   - Summary updates automatically as you filter

## üêõ Troubleshooting

### Common Issues

#### Page Not Loading

**Symptoms**: Blank page or error message

**Solutions**:
1. Refresh the page (F5 or Ctrl+R)
2. Clear browser cache
3. Check internet connection
4. Try a different browser
5. Check if the site URL is correct

#### Data Not Displaying

**Symptoms**: Empty tables or "No data available" message

**Solutions**:
1. Check if filters are too restrictive - try clearing filters
2. Verify you're on the correct page
3. Wait a few seconds for data to load
4. Refresh the page
5. Check browser console for errors (F12)

#### Map Not Showing

**Symptoms**: Blank map or "Loading..." message persists

**Solutions**:
1. Check internet connection
2. Ensure JavaScript is enabled in browser
3. Try refreshing the page
4. Check if pop-up blockers are interfering
5. Verify coordinates exist in the data

#### Filters Not Working

**Symptoms**: Filters don't update results

**Solutions**:
1. Ensure all required filters are selected (e.g., Region before Province)
2. Clear all filters and try again
3. Refresh the page
4. Check browser console for errors

#### Search Returns No Results

**Symptoms**: "No results found" message

**Solutions**:
1. Try broader search terms
2. Check spelling
3. Remove special characters
4. Try searching in specific sections instead of global search

#### Export Not Working

**Symptoms**: CSV file doesn't download

**Solutions**:
1. Check browser download settings
2. Allow pop-ups for the site
3. Check if browser blocks downloads
4. Try a different browser
5. Check available disk space
