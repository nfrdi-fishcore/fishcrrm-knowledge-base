Information and resources for the FishCRRM 1.1 Knowledge Base

About this Knowledge Base

The FishCRRM 1.1 Knowledge Base is a comprehensive digital platform designed to centralize and organize information, resources, and data related to the Fisheries and Coastal Resilient Resource Management (FishCRRM) Sub-component 1.1 of the Philippine Fisheries and Coastal Resiliency (FishCoRe) Project. This seven-year initiative (2023â€“2029) aims to improve fisheries management, rehabilitate marine habitats, modernize aquaculture, and strengthen the livelihoods of over 1.15 million fisherfolk across 11 regions and 24 provinces.

This knowledge base serves as a centralized repository for project documentation, implementation structures, contact directories, activity records, reference materials, and coverage area information. It provides stakeholders, project personnel, and partners with easy access to essential project information and resources through an intuitive, user-friendly interface.

The platform is built using modern web technologies and integrates with Google Sheets for real-time data management, ensuring that information is always up-to-date and accessible to authorized users. All data is securely managed and synchronized automatically, providing a seamless experience for users across different devices and locations.

How to Use This Knowledge Base

Navigation

Use the top navigation bar to access different sections of the knowledge base. The main sections include: Implementation Structure (organizational hierarchy), FMA Profile (characteristics and measurements for FMA 06 and FMA 09), FMA Municipalities (comprehensive municipality listings), Landing Centers Map (interactive geographic visualization), Activities (timeline of conducted activities), Directory (contact information), and References (document repository). You can also use the Quick Links on the home page for faster access to key sections.

Global Search

The global search bar in the navigation allows you to search across all data and content in the knowledge base. Simply type your search term and press Enter or click the search icon. The search function looks through all sheets including implementation structure, municipalities, activities, directories, and references. Results are ranked by relevance, and you can click on any result to navigate directly to the relevant section.

Filters and Advanced Search

Each section includes specialized filters to help you find specific information quickly. FMA Municipalities: Filter by FMA, Region, and Province (province options depend on selected region). Use pagination controls to navigate through large datasets. Landing Centers Map: Filter by FMA, Region, Province, Fishing Ground, or search by city/municipality or landing center name. Filters update markers in real-time. Activities: Filter by year and search by title, location, or resource person. Directory: Filter by component, employment type, or search by name, position, or email.

Export and Data Management

Export directory information as CSV files for offline use or further analysis. The export function includes all filtered results, allowing you to export specific subsets of data. CSV files can be opened in Excel, Google Sheets, or any spreadsheet application for additional analysis, reporting, or record-keeping purposes.

Mobile Responsive Design

The knowledge base is fully responsive and optimized for use on desktop computers, tablets, and mobile devices. All features are accessible on mobile devices, with touch-friendly interfaces and adaptive layouts that adjust to different screen sizes. The Landing Centers map includes mobile-optimized controls and filter panels.

Key Features

Implementation Structure

View the complete organizational structure of the FishCRRM Component at National (NPMO), FMA (RPIU), and Regional (FCU) levels. Explore hierarchical relationships, component compositions, and key personnel across all organizational levels. The structure is dynamically loaded and organized by level for easy navigation.

FMA Profile

Access detailed characteristics and measurements for FMA 06 and FMA 09, including key statistics, geographic information, and management parameters. This section provides comprehensive profiles of each Fishery Management Area with organized data presentation and visual indicators.

FMA Municipalities

Comprehensive list of municipalities with advanced filtering capabilities. Features include: Multi-level filtering by FMA, Region, and Province (with dependent province selection), Pagination with customizable items per page (10, 20, 50, 100), Summary Statistics showing total municipalities and NSAP monitored count, Side-by-side tables displaying municipalities by Region (alphabetically sorted with BARMM at bottom) and by FMA (numerically sorted), and Real-time filtering that updates both the table and summary statistics simultaneously.

Landing Centers Map

Interactive map visualization powered by Leaflet.js with comprehensive features: Multiple filter options (FMA, Region, Province, Fishing Ground, and text search for city/municipality or landing center name), Base map controls allowing you to switch between different map tile layers (OpenStreetMap, Satellite, etc.), Real-time marker filtering that updates the map as you change filters, Info panel displaying count of visible markers, Collapsible filter sidebar for mobile-friendly access, Reset filters button for quick clearing, and Automatic bounds fitting to show all filtered markers. Only locations with valid coordinates are displayed on the map.

Activities

Browse activities, trainings, and coordination meetings with a visually appealing timeline view. Features include: Year filtering to view activities by specific year, Search functionality to find activities by title, location, or resource person, Chronological sorting with most recent activities first, Detailed activity cards showing date, location, resource person, and reference documents, and Responsive timeline layout optimized for all screen sizes.

Directory

Access comprehensive contact information for project personnel across three categories: Internal Directory (NFRDI FishCore personnel), External Directory (external partners and stakeholders), and NPMO Directory (National Project Management Office personnel). Features include: Tab-based navigation between directory types, Advanced filtering by component, employment type, office, and FMA lead, Search functionality across names, positions, and email addresses, CSV export for filtered results, and Detailed contact cards with all relevant information.

References

Download reference documents and materials organized by category. Features include: Category-based organization for easy browsing, Direct download links to reference files, Search functionality to find specific documents, Document metadata including titles and categories, and Secure file access through Google Sheets integration.

Data Sources and Technology

The FishCRRM 1.1 Knowledge Base is built on modern web technologies and integrates seamlessly with Google Sheets for data management. All data is stored in Google Sheets, which serves as the backend database, allowing for real-time updates and collaborative data management.

Data Management: Data is organized across multiple Google Sheets, each corresponding to a specific section of the knowledge base. Authorized personnel can update data directly in Google Sheets, and changes are automatically reflected in the knowledge base without requiring code changes or redeployment.

Technology Stack: The platform uses a Single Page Application (SPA) architecture with client-side routing, ensuring fast navigation and smooth user experience. The interface is built with Bootstrap 5 for responsive design, Leaflet.js for interactive mapping, and modern JavaScript (ES6+) for dynamic functionality. All features are optimized for performance and accessibility.
